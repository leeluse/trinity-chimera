# Environment Variables

## 필수
- `SUPABASE_URL`
- `SUPABASE_KEY`

## Frontend API
- `NEXT_PUBLIC_API_URL`
- `BACKEND_API_URL`
- `NEXT_PUBLIC_API_FETCH_TIMEOUT_MS`

## LLM (Chat/Ollama)
- `LLM_PROVIDER`
- `OLLAMA_BASE_URL`
- `OLLAMA_MODEL`
- `LLM_TIMEOUT`

## LLM (Evolution / OpenAI-Compatible)
- `OPENAI_BASE_URL`
- `OPENAI_API_KEY`
- `OPENAI_MODEL`
- `ANTHROPIC_BASE_URL`
- `ANTHROPIC_API_KEY`
- `ANTHROPIC_MODEL`
- `EVOLUTION_LITELLM_ENABLE` (1/0: LiteLLM SDK 경로 사용 여부)
- `EVOLUTION_LITELLM_PROVIDER` (예: `openai`, `anthropic`)
- `EVOLUTION_LLM_FALLBACK_MODELS` (쉼표 구분 fallback 모델 목록)
- `EVOLUTION_LLM_TIMEOUT_SECONDS`
- `EVOLUTION_LLM_REQUEST_RETRIES`
- `EVOLUTION_LLM_RETRY_BACKOFF_SECONDS`
- `EVOLUTION_LLM_MAX_TOKENS`
- `EVOLUTION_LLM_MAX_CONCURRENCY`

## Evolution Scheduler
- `EVOLUTION_POLL_MINUTES`
- `EVOLUTION_HEARTBEAT_MINUTES`
- `EVOLUTION_FORCE_MODE`

## Tunnel / Runtime
- `BACKEND_PORT`
- `TUNNEL_SUBDOMAIN`
- `LT_LOCAL_HOST`
- `LT_HOST`
