# 🛠 Agent Toolbox: 기술 상세 명세

에이전트들이 전략 수립을 위해 자율적으로 선택하고 조합할 수 있는 핵심 도구들의 명세입니다.

## 1. HMM Regime Detector (시장 국면 감지기)
- **목적**: 비정상성(Non-stationarity)을 가진 금융 데이터에서 현재 시장의 '상태'를 확률적으로 분류.
- **입력 피처**: 로그 수익률, 실현 변동성(20봉), 거래량 프로파일.
- **출력**: 
  - `Regime 0 (Bear)`: 높은 변동성 + 음의 평균 수익률.
  - `Regime 1 (Sideways)`: 낮은 변동성 + 0에 수렴하는 평균 수익률.
  - `Regime 2 (Bull)`: 적정 변동성 + 양의 평균 수익률.
- **에이전트 활용**: 에이전트는 현재 Regime 정보를 바탕으로 자신의 전략 가중치를 조정함.

## 2. Triple Barrier Labeler (지능형 성과 평가기)
- **목적**: 단순 미래 수익률 대신, 실제 트레이딩의 세 가지 제약 조건을 반영한 품질 높은 학습 데이터 생성.
- **3개 장벽(Barriers)**:
    1. **TP Barrier (Upper)**: 목표 수익률 도달 시 '성공' 레이블.
    2. **SL Barrier (Lower)**: 손절가 도달 시 '실패' 레이블.
    3. **Time Barrier (Vertical)**: 시간 만료 시 현재 수익률을 바탕으로 레이블 결정.
- **에이전트 활용**: 에이전트의 모델이 '성공' 레이블을 더 잘 예측하도록 스스로 목적 함수를 조정.

## 3. Dynamic Feature Injector (동적 피처 주입기)
- **목적**: 기술적 지표 외에 에이전트에게 필요한 추가 무기 제공.
- **제공 데이터**: RSI, MACD, Bollinger Bands, Volume Profile, Orderbook Imbalance.
- **자율성**: 에이전트는 이 중 어떤 피처가 현재 수익률과 높은 상관관계를 갖는지 스스로 계산하여 입력값으로 선택함.
