# 🧠 Agent Autonomy: 자율 피봇 및 학습 메커니즘

에이전트가 스스로의 전략을 의심하고 새로운 최적점을 찾아가는 '자율적 판단' 시스템에 대한 명세입니다.

## 1. 품질 모니터링 (Quality Monitoring)
- **에이전트 자기 평가**: 에이전트는 자신의 예측 확률과 실제 시장 결과 간의 괴리(Calibration Error)를 상시 모니터링함.
- **피봇 시그널 (Pivot Signal)**: 최근 20회 매매의 승률이 기대 승률보다 30% 이상 낮아질 경우 '전략 재수립' 모드 진입.

## 2. 전략 탐색 (Strategic Search)
- **하이퍼파라미터 자율 튜닝**: 에이전트는 성과 저하시 스스로 학습 파라미터(Learning Rate, Depth 등)를 변경하며 백테스팅을 재수행함.
- **도구 재조합**: HMM의 Regime 분류가 현재 유효하지 않다고 판단하면, 다른 피처(예: 오더북 데이터)의 비중을 높여 새로운 모델을 독자적으로 구축함.

## 3. 자가 진화 (Evolutionary Evolution)
- **Shadow Experiment**: 에이전트는 실매매를 하면서도 백그라운드에서 끊임없이 3~4개의 '가상 자아(Sub-strategy)'를 운영함.
- **자산 이전**: 가상 자아 중 성과가 더 좋은 모델이 발견되면 에이전트는 자신의 메인 엔진을 해당 모델로 즉시 교체함.
