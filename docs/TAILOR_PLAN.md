## 기술적 상세 계획

### Triple Barrier Labeler
- **알고리즘:** 수익 목표(TP), 손절(SL), 시간 장벽 기반 레이블링
- **라이브러리:** pandas, numpy, scikit-learn
- **입력:** OHLCV 데이터 + 기술적 지표 + HMM regime 정보
- **출력:** ML 학습용 레이블 {-1, 0, 1} 및 가중치 [0.0-1.0]
- **Barrier 설정:** 
  - TP: 2.0x ATR (Average True Range)
  - SL: 1.0x ATR  
  - 시간 장벽: 20개 봉 (최대 보유 기간)
- **통합:** HMM regime별 barrier 조정 (Bull: aggressive, Bear: conservative)

### RL 학습 시스템
- **알고리즘:** PPO (Proximal Policy Optimization) + SAC (Soft Actor-Critic)
- **라이브러리:** Stable-Baselines3, PyTorch
- **환경:** Gymnasium 트레이딩 환경 (John 개발)
- **학습 설정:**
  - 배치 크기: 64-256
  - 학습률: 3e-4 (Adam optimizer)
  - 감가율: 0.99
  - 에피소드 길이: 1000 steps
- **최적화:** Optuna를 이용한 하이퍼파라미터 최적화
- **모니터링:** TensorBoard 로깅 및 체크포인팅
